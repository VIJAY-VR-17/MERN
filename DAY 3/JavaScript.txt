JavaScript

JavaScript is a high-level, interpreted scripting language used to make web pages interactive and dynamic. It works with HTML and CSS to create responsive and user-friendly websites.

Features of JavaScript

Lightweight and interpreted language
Object-oriented programming support
Event-driven programming
Platform independent
Dynamically typed
Case-sensitive language



Adding JavaScript to HTML

JavaScript can be added to an HTML document in three ways.


1. Inline JavaScript

JavaScript is written inside an HTML element using event attributes.

<button onclick="alert('Hello World!')">Click Me</button>


2. Internal JavaScript

JavaScript code is written inside the <script> tag.

<!DOCTYPE html>
<html>
<head>
    <title>Internal JavaScript</title>
</head>
<body>

<h2>JavaScript Example</h2>

<script>
    document.write("Welcome to JavaScript");
</script>

</body>
</html>



3. External JavaScript

JavaScript is written in a separate file with the .js extension.

HTML File
<script src="script.js"></script>


script.js
document.write("External JavaScript File");



JavaScript Output Methods


document.write()

Displays output directly on the web page.

document.write("Hello World");



console.log()

Displays output in the browser console.

console.log("Hello World");



alert()

Displays output in a popup alert box.

alert("Welcome");

Variables

Variables are containers used to store data.

JavaScript provides three keywords to declare variables.

var
let
const


var

Can be redeclared
Can be reassigned
Function scoped

var name = "John";

console.log(name);

name = "David";

console.log(name);

Output

John
David



let

Cannot be redeclared
Can be reassigned
Block scoped

let age = 20;

age = 21;

console.log(age);

Output

21



Constants

Constants are variables whose values cannot be changed after initialization.

Constants are declared using the const keyword.


Features

Cannot be redeclared
Cannot be reassigned
Block scoped


const PI = 3.14;

console.log(PI);

Output

3.14

Invalid Example

const PI = 3.14;

PI = 3.14159;

Output

TypeError




Data Types

JavaScript data types are classified into two categories.

Primitive Data Types
Reference (Non-Primitive) Data Types



Primitive Data Types

Primitive data types store a single value.

Types of Primitive Data Types

Number
String
Boolean
Undefined
Null
BigInt
Symbol


Number
let num = 100;

console.log(num);

Output

100



String

let name = "Rahul";

console.log(name);

Output

Rahul



Boolean
let result = true;

console.log(result);

Output

true



Undefined

A variable declared without a value.

let x;

console.log(x);

Output

undefined



Null

Represents an intentional empty value.

let value = null;

console.log(value);

Output

null



BigInt

Used to store very large integers.

let number = 123456789012345678901234567890n;

console.log(number);



Symbol

Represents a unique identifier.

let id = Symbol("id");

console.log(id);

Output

Symbol(id)



Reference (Non-Primitive) Data Types

Reference data types store collections of data or objects.

Examples

Object
Array
Function
Date
Map
Set



Object

let student = {
    name: "John",
    age: 20
};

console.log(student.name);

Output

John



Array

let marks = [80, 90, 95];

console.log(marks[1]);

Output

90



Function

function add(a, b) {
    return a + b;
}

console.log(add(10, 20));

Output

30



Difference Between Primitive and Reference Data Types


Primitive Data Type	Reference Data Type

Stores actual value	Stores memory reference
Immutable	        Mutable
Compared by value	Compared by reference
Fixed size	        Dynamic size

Primitive Example

let a = 10;
let b = a;

b = 20;

console.log(a);
console.log(b);

Output

10
20

Reference Example

let obj1 = { name: "John" };

let obj2 = obj1;

obj2.name = "David";

console.log(obj1.name);

Output

David





Operators

Operators are symbols used to perform operations on variables and values.

Types of Operators

Arithmetic Operators
Assignment Operators
Comparison Operators
Logical Operators
Increment and Decrement Operators
Ternary Operator
Type Operator
Bitwise Operators



Arithmetic Operators
Operator	Description
+	Addition
-	Subtraction
*	Multiplication
/	Division
%	Modulus
**	Exponentiation

Example

let a = 10;
let b = 3;

console.log(a + b);
console.log(a - b);
console.log(a * b);
console.log(a / b);
console.log(a % b);
console.log(a ** b);





Assignment Operators
Operator	Example
=	x = 5
+=	x += 5
-=	x -= 5
*=	x *= 5
/=	x /= 5
%=	x %= 5

Example

let x = 10;

x += 5;

console.log(x);

Output

15



Comparison Operators

Operator	Description
==	Equal to
===	Strict Equal
!=	Not Equal
!==	Strict Not Equal
>	Greater Than
<	Less Than
>=	Greater Than or Equal
<=	Less Than or Equal

Example

console.log(10 == "10");
console.log(10 === "10");  //compares data type also

Output

true
false



Logical Operators

Operator	Description
&&	Logical AND
||	Logical OR
!	Logical NOT

Example

let age = 20;

console.log(age >= 18 && age <= 60);
console.log(age < 18 || age > 60);
console.log(!(age < 18));



Increment and Decrement Operators

Increment

let x = 5;

x++;

console.log(x);

Output

6

Decrement

let y = 5;

y--;

console.log(y);

Output

4



Ternary Operator

Syntax

condition ? expression1 : expression2;

Example

let age = 18;

let result = age >= 18 ? "Eligible" : "Not Eligible";

console.log(result);

Output

Eligible



Type Operator

typeof

Returns the data type of a variable or value.

console.log(typeof 10);
console.log(typeof "Hello");
console.log(typeof true);

Output

number
string
Boolean



Bitwise Operators
Operator	Description
&	Bitwise AND
|	Bitwise OR
^	Bitwise XOR
~	Bitwise NOT
<<	Left Shift
>>	Right Shift

Example

console.log(5 & 3);
console.log(5 | 3);
console.log(5 ^ 3);

Output

1
7
6

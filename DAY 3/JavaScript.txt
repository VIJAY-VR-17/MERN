JavaScript

JavaScript is a high-level, interpreted scripting language used to make web pages interactive and dynamic. It works with HTML and CSS to create responsive and user-friendly websites.

Features of JavaScript

Lightweight and interpreted language
Object-oriented programming support
Event-driven programming
Platform independent
Dynamically typed
Case-sensitive language



Adding JavaScript to HTML

JavaScript can be added to an HTML document in three ways.


1. Inline JavaScript

JavaScript is written inside an HTML element using event attributes.

<button onclick="alert('Hello World!')">Click Me</button>


2. Internal JavaScript

JavaScript code is written inside the <script> tag.

<!DOCTYPE html>
<html>
<head>
    <title>Internal JavaScript</title>
</head>
<body>

<h2>JavaScript Example</h2>

<script>
    document.write("Welcome to JavaScript");
</script>

</body>
</html>



3. External JavaScript

JavaScript is written in a separate file with the .js extension.

HTML File
<script src="script.js"></script>


script.js
document.write("External JavaScript File");

1. Type Conversion

Type Conversion is the process of converting a value from one data type to another.

There are two types of type conversion:

Implicit Type Conversion (Automatic)
Explicit Type Conversion (Manual)
Implicit Type Conversion

JavaScript automatically converts one data type into another during an operation.

Example
let a = "10";
let b = 5;

console.log(a + b);   // "105"
console.log(a - b);   // 5

Explanation:

+ performs string concatenation when one operand is a string.
-, *, and / convert strings to numbers automatically.
Explicit Type Conversion

The programmer manually converts one data type to another.

Number()

Converts a value into a number.

let age = Number("20");

console.log(age);

Output

20
String()

Converts a value into a string.

let num = 100;

let text = String(num);

console.log(text);

Output

"100"
Boolean()

Converts a value into a Boolean.

console.log(Boolean(1));
console.log(Boolean(0));

Output

true
false

4. Conditional Statements

Conditional statements execute different blocks of code based on a condition.

Types:

if
if...else
else if Ladder
switch
Ternary Operator
if Statement

Executes code only when the condition is true.

let age = 20;

if(age >= 18){
    console.log("Eligible to Vote");
}
if...else Statement
let age = 15;

if(age >= 18){
    console.log("Eligible");
}
else{
    console.log("Not Eligible");
}
else if Ladder

Used when multiple conditions need to be checked.

let mark = 82;

if(mark >= 90){
    console.log("Grade A+");
}
else if(mark >= 80){
    console.log("Grade A");
}
else{
    console.log("Grade B");
}
switch Statement

Used to select one block from multiple cases.

let day = 2;

switch(day){
    case 1:
        console.log("Monday");
        break;

    case 2:
        console.log("Tuesday");
        break;

    default:
        console.log("Invalid Day");
}
Ternary Operator

A shorthand form of if...else.

Syntax
condition ? expression1 : expression2;
Example
let age = 18;

let result = age >= 18 ? "Adult" : "Minor";
5. Loops

Loops execute a block of code repeatedly until a condition becomes false.

Types:

for Loop
while Loop
do...while Loop
for Loop

Used when the number of iterations is known.

Syntax
for(initialization; condition; increment){
    statements;
}
Example
for(let i=1;i<=5;i++){
    console.log(i);
}
while Loop

Checks the condition before execution.

Syntax
while(condition){
    statements;
}
Example
let i=1;

while(i<=5){
    console.log(i);
    i++;
}
do...while Loop

Executes the loop body at least once.

Syntax
do{
    statements;
}
while(condition);
Example
let i=1;

do{
    console.log(i);
    i++;
}
while(i<=5);

Example

8. Scope

Scope determines where a variable can be accessed.

Types:

Global Scope
Local Scope (Function Scope)
Block Scope
Global Scope

Variables declared outside all functions are accessible everywhere.

let name = "Rahul";

function display(){

    console.log(name);

}

display();
Local Scope

Variables declared inside a function can only be accessed within that function.

function show(){

    let age = 20;

    console.log(age);

}

Outside the function:

console.log(age);

Output

ReferenceError
Block Scope

Variables declared using let and const inside a block {} are accessible only within that block.

if(true){

    let x = 10;

    console.log(x);

}

Outside the block:

console.log(x);

Output

ReferenceError








Arrays

An Array is a collection of multiple values stored in a single variable. Each value in an array is called an element, and every element has an index starting from 0.

Features of Arrays
Stores multiple values in a single variable.
Elements can be of different data types.
Arrays are dynamic; their size can change.
Index starts from 0.
Syntax
let arrayName = [value1, value2, value3];
Example
let fruits = ["Apple", "Banana", "Orange"];

console.log(fruits);
console.log(fruits[0]);
console.log(fruits[2]);

Output

["Apple", "Banana", "Orange"]
Apple
Orange
